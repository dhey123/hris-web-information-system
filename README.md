# HRIS Web Information System

## Overview
Human Resource Information System (HRIS) developed during an internship program to support employee administration, compliance monitoring, recruitment processes, and operational management.

## Features
* Employee Data Management
* Leave Recapitulation Management
* Recruitment Management
* SHE (Safety, Health & Environment) Management
* Legal Management
* Labour & Government Compliance Management
* General Affair Management
* Master Data Management
* Dashboard & Reporting

## Tech Stack
* Laravel
* PHP
* MySQL
* JavaScript
* Tailwind CSS
